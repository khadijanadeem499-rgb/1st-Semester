**📚 1st Semester Study Hub**

Welcome to the 1st Semester Study Hub — a centralized platform for organizing and accessing all your semester notes, past papers, and study resources in one place.

This project is designed to help students quickly revise topics, download materials, and stay organized throughout the semester.

**🌟 Features**

**📂 Organized by Subject**

Easily navigate through different courses and topics.

**📄 Downloadable PDFs**

Access lecture notes, summaries, and study guides instantly.

**📱 Responsive Design**

Fully mobile-friendly for studying on the go.

**🔍 Search Functionality**

Quickly find topics using keywords.

**📂 Folder Structure**

├── index.html          # Main landing page
├── assets/             # Images, CSS, and JavaScript files
├── subjects/
│   ├── math-112/       # Calculus I notes
│   ├── physics/        # Mechanics and Waves

└── README.md           # Project documentation
